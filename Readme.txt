# Collection Dashboard

Welcome to the **Collection Dashboard**, an interactive Streamlit application designed for data analysis and visualization in Python. This dashboard provides users with intuitive insights into their collections, enabling better decision-making and management.

## Features
- **Interactive Visualizations**: Utilize various charts and graphs to explore your data.
- **User-Friendly Interface**: Simple navigation to access different analysis tools.
- **Data Insights**: Generate meaningful insights from your collections with ease.

## Setup Environment
To get started with the Collection Dashboard, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/achadansori/analisa_data_python
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd analisa_data_python
   ```

3. **Install Required Packages**:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Streamlit App
Once the setup is complete, you can launch the application with the following command:

```bash
streamlit run dashboard.py
```

Open your web browser and navigate to `http://localhost:8501` to view the dashboard.

## Usage
Upon launching the app, users will be greeted with a clean interface. Users can:
- Upload their data files.
- Select various analysis options.
- Visualize and interpret their collection data seamlessly.
